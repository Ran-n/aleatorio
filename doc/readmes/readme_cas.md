### Escripts
#### ffix
Arregla los ficheros para que tan solo contengan los caracteres permitidos segun el tipo de dice

#### d4
Para sistemas cuaternarios.

#### d20
Para sistemas vigesimales.

#### d100
Para sistemas centesimales.
